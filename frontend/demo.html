你好！这是一个仿照 ChatGPT 风格设计的聊天界面，具有以下特点：


🎨 界面设计：

• 深色主题，护眼舒适

• 现代化的UI设计

• 流畅的动画效果


💬 聊天功能：

• 支持实时对话

• Markdown 格式渲染

• 流式响应显示


📱 响应式设计：

• 支持桌面和移动设备

• 自适应屏幕尺寸


代码示例：

// JavaScript 示例
function sendMessage() {
    const content = messageInput.value.trim();
    if (!content) return;
    
    addMessageToUI({
        role: 'user',
        content: content
    });
}

支持哪些技术栈？

这个前端项目使用了以下技术栈：


前端技术：

• HTML5 - 语义化标记

• CSS3 - 现代样式和动画

• Vanilla JavaScript - 原生JS实现

• Font Awesome - 图标库


后端对接：

• RESTful API

• Server-Sent Events (流式响应)

• JWT 身份验证


这是一个纯前端实现，无需复杂的构建工具，直接在浏览器中运行！✨
